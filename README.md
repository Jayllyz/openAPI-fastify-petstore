## Tech Stack

- Node.js
- TypeScript
- Fastify
- openAPI 3
- Prisma
- PostgreSQL
- Docker

## Run locally

To run these commands you need to install :
[task](https://taskfile.dev/#/installation)

```bash
# Start docker
task docker

# Install dependencies
task install

# Init database
task prisma

# Start server
task dev
```
